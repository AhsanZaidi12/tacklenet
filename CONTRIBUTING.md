# Contributing to TackleNet

Thank you for your interest in TackleNet.

## Bug reports and questions

Open an issue on GitHub. For dataset-specific questions, contact ahsanzaidi@ksu.edu.

## Pull requests

Pull requests fixing bugs in evaluation scripts or improving documentation are welcome.
Please open an issue first to discuss proposed changes.

## Use restrictions

This repository is a research benchmark release. The dataset and associated code are
intended for sports safety and computer vision research only.

- Do not use TackleNet for player identification, surveillance, or any identity-related task.
- Do not use trained models as replacements for expert medical or safety evaluation.
